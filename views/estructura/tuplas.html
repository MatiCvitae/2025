¿Qué es una tupla?

  1. Una tupla es una secuencia ordenada e inmutable de elementos.
  2. Los elementos están en un orden que no cambia.
  3. No se pueden modificar una vez creada.
  4. Pueden contener diferentes tipos de datos (números, texto, etc.).
  5. Ejemplo en Python: (1, "hola", 3.14)
  6. Se accede a sus elementos por índice: mi_tupla[1] devuelve "hola".
  7. A diferencia de una lista, que se define con [], la tupla se define con ().

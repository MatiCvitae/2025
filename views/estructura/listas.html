Lista

  1. Una Lista es una colección ordenada de elementos pero en el contexto de la programación, una lista es una estructura de datos que permite almacenar múltiples elementos de cualquier tipo, ya sean numéricos textos numéricos objetos o otras listas

Métodos que se pueden usar en una lista

  1. En Python, las listas tienen varios métodos útiles que puedes aplicar para manipular sus elementos. Cada método tiene un propósito específico para trabajar con los elementos de una lista.

{{=metodo1}}

Lista original: ["manzana", "banana"]

Después de aplicar append("naranja"):

Resultado: {{=frutas1}} {{=type(frutas1)}}

frutas = ["manzana", "banana"] frutas.append("naranja") # Resultado: ["manzana", "banana", "naranja"]

{{=metodo2}}

Lista original: ["manzana", "banana"]

Después de aplicar extend(["naranja", "kiwi"]):

Resultado: {{=frutas2}} {{=type(frutas2)}}

frutas = ["manzana", "banana"] frutas.extend(["naranja", "kiwi"]) # Resultado: ["manzana", "banana", "naranja", "kiwi"]

{{=metodo3}}

Lista original: ["manzana", "banana"]

Después de aplicar insert(1, "naranja"):

Resultado: {{=frutas3}} {{=type(frutas3)}}

frutas = ["manzana", "banana"] frutas.insert(1, "naranja") # Resultado: ["manzana", "naranja", "banana"]

{{=metodo4}}

Lista original: ["manzana", "banana", "naranja"]

Después de aplicar remove("banana"):

Resultado: {{=frutas4}} {{=type(frutas4)}}

frutas = ["manzana", "banana", "naranja"] frutas.remove("banana") # Resultado: ["manzana", "naranja"]

{{=metodo5}}

Lista original: ["manzana", "banana", "naranja"]

Después de aplicar pop(1):

Elemento extraído: {{=fruta_extraida}}

Lista resultante: {{=frutas5}} {{=type(frutas5)}}

frutas = ["manzana", "banana", "naranja"] fruta = frutas.pop(1) # fruta = "banana" # frutas = ["manzana", "naranja"]

{{=metodo6}}

Lista: ["manzana", "banana", "naranja"]

Después de aplicar index("banana"):

Índice de "banana": {{=indice}}

Lista: {{=frutas6}} {{=type(frutas6)}}

frutas = ["manzana", "banana", "naranja"] indice = frutas.index("banana") # indice = 1

{{=metodo7}}

Lista: ["manzana", "banana", "banana", "naranja"]

Después de aplicar count("banana"):

Cantidad de "banana": {{=cantidad}}

Lista: {{=frutas7}} {{=type(frutas7)}}

frutas = ["manzana", "banana", "banana", "naranja"] cantidad = frutas.count("banana") # cantidad = 2

{{=metodo8}}

frutas = ["naranja", "banana", "manzana"] frutas.sort() # Resultado: ["banana", "manzana", "naranja"]

{{=metodo9}}

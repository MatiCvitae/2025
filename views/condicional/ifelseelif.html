Explicación del Código

Este ejemplo muestra cómo comparar dos números en Python usando una estructura condicional.

numero1 = 20 y numero2 = 20 definen dos variables numéricas.

iguales = (numero1 == numero2) guarda True o False dependiendo si los dos números son iguales.

Luego, el programa evalúa:

  • if numero1 < numero2: si es menor, muestra un mensaje diciendo eso.
  • elif iguales: si son iguales, muestra otro mensaje.
  • else: si no se cumple lo anterior, significa que es mayor.

El mensaje se guarda en la variable resultado usando .format(), que es una forma de insertar variables en un texto en Python 2.7.

Finalmente, todos estos datos se envían a esta página desde el controlador para que puedas verlos en tu navegador.
